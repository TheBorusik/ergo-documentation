```
Каждый раздел-этап имеет СВОЙ цвет и иконку, одинаковые ВЕЗДЕ:
левое меню, вкладки (иконки), доска (полоса + заголовок колонки),

Токены в tokens.css (менять здесь), потребители — lib/sections.ts:

  --section-supply    #4da3e0  снабжение — синий
  --section-approval  #d9a03f  согласования — янтарный

  --section-works     #e0823d  работы — оранжевый

  --section-positions #7d8fa3  позиции — серо-синий
Средняя насыщенность — читается на светлой и тёмной темах.
Пункты меню, не являющиеся этапами, переиспользуют гамму
(MENU_COLORS в lib/sections.ts) + розовый #d4537e для Продаж.
``` — основа стилей нового приложения

Источник: ergo-old/app/globals.css (полная версия) + ergo-old/styles/globals.css.

★★ РЕШЕНИЕ (2026-07-15): из Ergo берём только ПАЛИТРУ (цвета, темы).
   РАЗМЕРЫ И ПЛОТНОСТЬ — эталон VS Code / Hub (плотный инструментальный UI):
   шрифт 13px (xs 11 / sm 12 / md 13 / lg 14 / xl 16), радиусы 2-3px,
   кнопки 26px (sm 22px), инпуты 24px, строки таблиц 28px (comfort 36px),
   шапка 38px, statusbar 22px С ЗАЛИВКОЙ БРЕНДОМ (как синяя полоса VS Code),
   меню 200px (строки 26px, заголовки групп КАПСОМ 11px letter-spacing),
   иконки 13-15px, user-select:none (текст в таблицах/полях — можно).
   VS Code-паттерны — эталон, при необходимости расширяем сами.
   Реализация: ergo-erp/src/styles/tokens.css + globals.css.
   Размеры таблиц/шрифтов ниже по тексту — УСТАРЕЛИ (были 0.875rem/40px).

═══════════════════════════════════════════════════
ПРИНЦИПЫ (как стилизованы экраны Ergo)
═══════════════════════════════════════════════════
```
1. ВСЕ цвета — через CSS-переменные (--color-*), НЕ хардкод.
   Компоненты пишут style={{ color: "var(--color-text-primary)" }}
   → темы переключаются без правки компонентов.
2. Тёмная тема = класс .dark на корне (переопределяет переменные).
3. Плотность таблиц = атрибут data-density="comfort" на body
   (по умолчанию компактно).
4. Стек: Tailwind CSS v4 (@import "tailwindcss") + shadcn/ui
   (button, card, alert-dialog, toast...) + lucide-react (иконки).
5. Утилитарные классы Tailwind для layout, переменные для цветов.
```

ТЕКСТ (светлая → тёмная):
  --color-text-primary:   #111827 → #e5e7eb
  --color-text-secondary: #6b7280 → #9ca3af
  --color-text-tertiary:  #9ca3af → #6b7280
