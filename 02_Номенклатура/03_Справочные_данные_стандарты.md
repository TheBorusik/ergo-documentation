## 1. Класс герметичности воздуховодов
**Стандарт:** EN 1507, EN 12237, СП 60.13330.2012

| symbol | name | описание |
|---|---|---|
| A | Класс A | Базовый, для жилых и офисных помещений. Потери ≤1,35 л/с/м² при 400 Па |
| B | Класс B | Повышенная герметичность, транзитные участки, шахты. Потери ≤0,45 л/с/м² |
| C | Класс C | Высокая герметичность, давление 1000–2000 Па, опасные вещества. Потери ≤0,15 л/с/м² |
| D | Класс D | Максимальная, по спецзаданию для критически важных систем |

**ГОСТ (РФ):** СНиП 41-01-2003 — два класса:
| symbol | name |
|---|---|
| Н | Нормальный (аналог A/B) |
| П | Плотный (аналог C/D) |

## 14. Что обновить в measurement_types и measurements

### Обновить measurement_type "Класс герметичности" (id 21)
Текущие значения A/B/C/D — правильные по EN 1507/EN 12237.
Добавить российские Н/П как отдельный тип:

```
Новый measurement_type:
44 | Класс герметичности РФ | false | Н (нормальный), П (плотный) по СНиП 41-01-2003
```

### Заменить "Класс фильтрации" (id 22)
Текущие G1-G4, M5-M6, F7-F9, H10-H14, U15 — смешаны два разных стандарта!

**Правильно разделить на:**
```
22 | Класс фильтрации EN 779  | false | G1-G4, M5-M6, F7-F9 (старый стандарт)
45 | Класс фильтрации ISO 16890| false| ISO Coarse, ISO ePM10, ISO ePM2.5, ISO ePM1
46 | Класс фильтрации EPA/HEPA | false| E10-E12, H13-H14, U15-U17 (EN 1822)
```

### Обновить "Класс прочности" (id 48)
Добавить полный ряд: 3.6, 4.6, 4.8, 5.6, 5.8, 6.8, **8.8**, 9.8, **10.9**, **12.9**

### Добавить новые measurement_types:
```
47 | Климатическое исполнение расширенное | false | У, УХЛ, ХЛ, Т, М, ТМ, О, В
48 | Условный диаметр DN                 | false | DN 15, DN 20... DN 300
49 | Условное давление PN                | false | PN 6, PN 10, PN 16, PN 25, PN 40
50 | Резьба метрическая                  | false | M3, M4, M5, M6, M8, M10...
```

### Добавить хладагент R717 и R744:
```
R717 (аммиак)  — промышленные холодильные системы
R744 (CO₂)     — транскритические системы
```
